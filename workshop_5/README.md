## Завдання:
1. Переглянути матеріал лекції, виконати практикум
2. Опанувати у Figma інструменти по додаванню і відніманню фігур(булеві групи)
3. Створити два значка за вибором у solid стилі, використовуючі булеві групи
4. Написати звіт
